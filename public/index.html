Helsinkihot

Why this kind of service?

We built HelsinkiHot because of three reasons:

  • There was no good web service for getting information about what's happening in Helsinki
  • We wanted to experiment with the cutting-edge web technologies and visualizations
  • We needed a project for a school course :)

HelsinkiHot is made entirely with web technologies

People behind this

HelsinkiHot uses some bleeding-edge web technologies (mainly canvas) and thus it is not accessible for ancient browsers.

Supported browsers include

  • Chrome 9+
  • Safari 3.2+
  • Firefox 3.5+
  • Internet Explorer 9
  • Mobile Safari 3.2+
  • Opera 10.6+

Note that Internet Explorer versions prior to 9 are not supported.
